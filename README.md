# HDRezka Downloader (Firefox)

Расширение для браузера Firefox, позволяющее скачивать фильмы и сериалы с сайта HDRezka и любых его зеркал (например, `rezka.fi`, `hdrezka.ag`, `rezka.cc` и т.д.).

Расширение "перехватывает" прямые ссылки на видео в формате MP4 для разных вариантов качества (360p, 480p, 720p, 1080p) и предоставляет удобные кнопки для их скачивания.

## Особенности
- **Поддержка любых зеркал HDRezka**: Расширение работает на любых доменах, содержащих `rezka` или `hdrezka`.
- **Прямое скачивание MP4**: Вы можете скачивать файлы напрямую средствами браузера.
- **Поддержка сериалов**: При переключении серии в плеере ссылки автоматически обновляются в расширении.
- Разработано специально под архитектуру браузера Firefox (использует `filterResponseData`).

## Как установить расширение в Firefox

Так как расширение не загружено в официальный магазин Mozilla Add-ons, вы можете установить его в режиме разработчика локально:

1. Скачайте или клонируйте этот репозиторий на свой компьютер:
   ```bash
   git clone https://github.com/ВАШ_НИК/hdrezka-downloader.git
   ```
2. Откройте браузер **Firefox**.
3. В адресной строке введите `about:debugging` и нажмите Enter.
4. В левом меню выберите раздел **Этот Firefox** (This Firefox).
5. Нажмите на кнопку **Загрузить временное дополнение...** (Load Temporary Add-on...).
6. В открывшемся окне перейдите в папку с расширением и выберите файл `manifest.json`.
7. Расширение установлено! В панели инструментов (справа вверху) появится иконка. Рекомендуется закрепить её.

*(Примечание: Временные дополнения в Firefox удаляются после перезапуска браузера. Для постоянного использования расширение нужно либо каждый раз загружать заново, либо запаковать и подписать через Mozilla Developer Hub).*

## Как использовать

1. Зайдите на любое рабочее зеркало HDRezka (например, `rezka.fi`).
2. Откройте страницу нужного фильма или сериала.
3. **Обязательно запустите плеер!** Нажмите на кнопку Play, чтобы видео начало загружаться.
4. Нажмите на иконку расширения в панели браузера.
5. Появятся кнопки с вариантами качества видео (например: `[720p] Скачать (MP4)`).
6. Кликните по кнопке, и файл начнет скачиваться!

## Разработка

Расширение использует:
- `manifest.json` (Manifest V3)
- `background.js` (перехват сетевых AJAX-запросов плеера)
- `content.js` (поиск ссылок на странице)
- `popup.html / popup.js` (пользовательский интерфейс)
